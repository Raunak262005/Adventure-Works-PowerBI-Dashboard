# Adventure Works Sales Dashboard | Power BI Project

## Project Overview
Designed and developed an interactive Power BI dashboard using the Adventure Works dataset to analyze business performance, sales trends, customer insights, and product profitability.

This project demonstrates practical skills in data analysis, dashboard design, data modeling, and business intelligence reporting.

---

## Key Business Insights
- Identified top-performing products and categories
- Analyzed yearly sales and profit trends
- Tracked KPIs including Total Sales, Profit, and Orders
- Built customer and regional performance analysis
- Created interactive visualizations for better decision-making
